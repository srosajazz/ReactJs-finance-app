This project was made using the follow technologies:

* [Typescript](https://www.typescriptlang.org/)

* [Styled Components](https://styled-components.com/)

# :computer: How to run

```bash
# Install Dependencies
$ yarn

# Run Aplication
$ yarn start
```
Go to http://localhost:3000/ to see the result.
